# ------------------------------
# Type 목록
# ✨ feat     : 새로운 기능 추가
# 🐛 fix      : 기능수정, 버그 수정
# 💡 chore    : 오타 수정 및 새로운 기능이 추가되지 않고, 코드가 변경 된 경우 (주석 추가 및 수정 포함)
# 📝 docs     : 문서 수정 (README 등)
# 🚚 build    : 빌드 관련 파일 수정
# ✅ test     : 테스트 코드 추가
# ♻️ refactor : 코드 리팩터링
# 🚑 hotfix   : 긴급 수정
# ------------------------------

# <Type>: <Subject>

# - 왜 변경했는지
